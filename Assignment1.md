# Assignment - 1
## Study Linux Commands

### 1. cat - concatenate files and print on the standard output

```
cat [OPTION]... [FILE]...
```

**Description**

Concatenate FILE(s) to standard output.
With no FILE, or when FILE is -, read standard input.
